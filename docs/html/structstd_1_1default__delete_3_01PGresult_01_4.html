std::default_delete< PGresult > Struct Reference

Default deleter for PGresult. More...

Detailed Description

There are two ways to specify deleter for std::unique_ptr

  1. By template parameter
  2. By "std::default_delete" template specialization

In the first case we lose the default constructor for the pointer. That's why the second way was chosen.
